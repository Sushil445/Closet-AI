# 👗 Closet-AI

Closet-AI is a smart clothing recommendation system that helps users choose the perfect outfit based on their **🧍‍♂️ gender**, **🎂 age**, **🧴 skin tone**, and **☀️ weather** (sunny, monsoon, or winter).  
The system intelligently scrapes online shopping platforms and generates outfit suggestions tailored to your preferences.

---

## 🚀 Features

- 🤖 **AI-powered** outfit recommendations
- 🌦️ **Weather-aware** clothing suggestions
- 🧍‍♂️🎂🧴 Personalization based on:
  - Gender
  - Age group
  - Skin tone
- 🛒 Live **data scraping** from shopping websites using **Selenium**
- 🧠 Uses **Gemini API** to generate intelligent search queries
- 👚 Simplifies online shopping with relevant fashion choices
